Functions

  • Format the data into a compatible format
  • Compress the data
  • Encrypt the data

Famous format

  • MKV(Matroska Video)
  • MPG(Motion Picture Experts Group)
  • MOV(QuickTime Video)
  • GIF(Graphics Interchange Format)
  • JPG(Joint Photographic Experts Group)
  • PNG(Portable Network Graphics)

Session Layer

  • Create and maintain dialogs between source and destination applications.

Email protocols

SMTP (Simple Mail Transfer Protocol)

  • Send the mail to the server
  • Server port 25

POP3 (Post Office Protocol)

  • Retrieve mail from the server
  • Server Passively listen port 110
  • Not store mail on the server

IMAP (Internet Message Access Protocol)

  • Retrieve mail from the server
  • Server Passively listen port 143
  • Leave a copy on the server

DNS (Domain Name System)

Types of resource record

  • A - An end device IPv4 address
  • NS - AN authoritative name server
  • AAAA - AN end device IPv6 address
  • MX - A mail exchange record
